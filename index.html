Algo Complexity
  • Algo is the step-by-step process to solve any problem.

Time complexity: Time complexity is mainly calculated by counting the number of steps to finish the execution.


Space complexity: Space complexity is the amount of space required to solve a problem. Space complexity = Auxiliary space + input size


Big O notation: Use to denote complexity
